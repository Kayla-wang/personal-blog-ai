# Node.js 后端开发

> 巩固后端开发能力，为 AI 应用开发打下基础

## 学习目标

- 掌握 Nest.js 框架的核心概念和最佳实践
- 理解 HTTP 协议与 RESTful API 设计
- 熟练使用 async/await 处理异步操作
- 掌握数据库操作与 ORM 使用

## 知识模块

### 1. Nest.js 框架

| 主题 | 内容 |
|------|------|
| 核心概念 | 模块、控制器、服务、依赖注入 |
| 装饰器 | @Module、@Controller、@Injectable |
| 中间件 | 请求处理、日志、认证 |
| 管道与守卫 | 数据验证、权限控制 |

### 2. HTTP 协议

| 主题 | 内容 |
|------|------|
| RESTful API | 资源设计、HTTP 方法、状态码 |
| WebSocket | 双向通信、实时更新 |
| SSE | 服务端推送、流式响应 |

### 3. 异步编程

| 主题 | 内容 |
|------|------|
| Promise | 链式调用、错误处理 |
| async/await | 同步风格的异步代码 |
| 事件循环 | 宏任务、微任务、执行顺序 |
| 并发控制 | Promise.all、Promise.race、限流 |

### 4. 数据库操作

| 主题 | 内容 |
|------|------|
| TypeORM | 实体定义、关系映射、查询构建 |
| Prisma | Schema 定义、类型安全、迁移 |
| PostgreSQL | SQL 基础、索引优化、事务 |
| Redis | 缓存策略、会话存储、消息队列 |

## 实践项目

构建一个完整的 RESTful API 服务：
- 用户认证（JWT）
- CRUD 操作
- 文件上传
- 定时任务
