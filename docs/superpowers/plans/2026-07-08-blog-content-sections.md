# 博客内容分区实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将博客分为「学习笔记」和「项目实践」两个独立板块，使用 Docusaurus docs 多实例实现。

**Architecture:** 使用 Docusaurus 的 docs 插件多实例功能，创建两个独立的文档区域。`docs/` 目录映射到 `/notes` 路径作为学习笔记，新建 `projects/` 目录映射到 `/projects` 路径作为项目实践。移除博客功能。

**Tech Stack:** Docusaurus 3.x, @docusaurus/plugin-content-docs, @easyops-cn/docusaurus-search-local

---

## 文件结构

| 操作 | 文件 | 职责 |
|------|------|------|
| 新增 | `projects/index.md` | 项目实践首页 |
| 新增 | `sidebarsProjects.js` | 项目实践侧边栏配置 |
| 修改 | `docusaurus.config.js` | 配置双 docs 实例、导航栏、搜索 |
| 删除 | `blog/` | 移除博客功能 |

### Task 3: 更新 Docusaurus 配置

**Files:**
- Modify: `docusaurus.config.js`

- [ ] **Step 1: 更新 docs preset 配置**

修改 `docusaurus.config.js` 中的 presets 部分，将 docs 的 `routeBasePath` 改为 `notes`，关闭 blog：

```javascript
presets: [
  [
    'classic',
    /** @type {import('@docusaurus/preset-classic').Options} */
    ({
      docs: {
        sidebarPath: './sidebars.js',
        routeBasePath: 'notes',
      },
      blog: false,
      theme: {
        customCss: './src/css/custom.css',
      },
    }),
  ],
],
```

- [ ] **Step 2: 添加 projects 插件**

在 `docusaurus.config.js` 中添加 plugins 配置（在 presets 之后，themes 之前）：

```javascript
plugins: [
  [
    '@docusaurus/plugin-content-docs',
    {
      id: 'projects',
      path: 'projects',
      routeBasePath: 'projects',
      sidebarPath: './sidebarsProjects.js',
    },
  ],
],
```

- [ ] **Step 3: 更新导航栏**

修改 `themeConfig.navbar.items`：

```javascript
navbar: {
  title: '学习笔记',
  items: [
    {
      type: 'docSidebar',
      sidebarId: 'tutorialSidebar',
      position: 'left',
      label: '学习笔记',
    },
    {
      to: '/projects',
      label: '项目实践',
      position: 'left',
    },
    {
      to: '/notes/tags',
      label: '标签',
      position: 'left',
    },
  ],
},
```

- [ ] **Step 4: 更新搜索插件配置**

修改 themes 中的搜索插件配置：

```javascript
themes: [
  [
    '@easyops-cn/docusaurus-search-local',
    /** @type {import("@easyops-cn/docusaurus-search-local").PluginOptions} */
    ({
      hashed: true,
      language: ['en', 'zh'],
      highlightSearchTermsOnTargetPage: true,
      explicitSearchResultPath: true,
      docsRouteBasePath: ['/notes', '/projects'],
      indexBlog: false,
      indexDocs: true,
    }),
  ],
],
```

- [ ] **Step 5: 提交**

```bash
git add docusaurus.config.js
git commit -m "feat: configure dual docs instances and update navbar"
```
