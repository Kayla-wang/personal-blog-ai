# 博客内容分区设计

## 概述

将博客分为两个独立板块：「学习笔记」和「项目实践」，使用 Docusaurus docs 多实例实现。

## 目标

- 学习笔记：存放理论知识、技术学习内容，按主题分类
- 项目实践：存放实际项目的完整记录，按项目分类
- 两个板块在导航栏有独立入口
- 移除现有博客功能

## 文件结构

```
D:\project\docs\
├── docs/                    # 学习笔记
│   ├── ai-agent/           # AI Agent 主题（现有内容）
│   │   ├── 01-python-basics/
│   │   ├── 02-llm-fundamentals/
│   │   └── ...
│   ├── frontend/           # 未来扩展：前端学习笔记
│   └── index.md            # 学习笔记首页

├── projects/                # 项目实践（新建）
│   ├── blog-system/        # 示例项目
│   │   ├── index.md
│   │   └── ...
│   └── index.md            # 项目实践首页
│
├── sidebars.js             # 学习笔记侧边栏
├── sidebarsProjects.js     # 项目实践侧边栏（新建）
└── docusaurus.config.js    # 配置两个 docs 实例
```

## URL 路径

| 板块 | 文件位置 | 访问路径 |
|------|----------|----------|
| 学习笔记 | `docs/ai-agent/...` | `/notes/ai-agent/...` |
| 项目实践 | `projects/blog-system/...` | `/projects/blog-system/...` |

## 导航栏

```
[Logo: 学习笔记]   [学习笔记]   [项目实践]   [标签]
```

- 学习笔记 → `/notes`
- 项目实践 → `/projects`
- 标签 → `/notes/tags`
- 移除「博客」入口

## 配置变更

### docusaurus.config.js

```javascript
presets: [
  [
    'classic',
    {
      docs: {
        path: 'docs',
        routeBasePath: 'notes',
        sidebarPath: './sidebars.js',
      },
      blog: false,
      theme: {
        customCss: './src/css/custom.css',
      },
    },
  ],
],

plugins: [
  [
    '@docusaurus/plugin-content-docs',
    {
      id: 'projects',
      path: 'projects',
      routeBasePath: 'projects',
      sidebarPath: './sidebarsProjects.js',
    },
  ],
],

themeConfig: {
  navbar: {
    title: '学习笔记',
    items: [
      {
        type: 'docSidebar',
        sidebarId: 'tutorialSidebar',
        position: 'left',
        label: '学习笔记',
      },
      {
        to: '/projects',
        label: '项目实践',
        position: 'left',
      },
      {
        to: '/notes/tags',
        label: '标签',
        position: 'left',
      },
    ],
  },
}
```

### sidebarsProjects.js（新建）

```javascript
module.exports = {
  projectsSidebar: [
    {

```

### 搜索插件更新

```javascript
themes: [
  [
    '@easyops-cn/docusaurus-search-local',
    {
      hashed: true,
      language: ['en', 'zh'],
      highlightSearchTermsOnTargetPage: true,
      explicitSearchResultPath: true,
      docsRouteBasePath: ['/notes', '/projects'],
      indexDocs: true,
      indexBlog: false,
    },
  ],
],
```

1. **创建项目实践目录**

2. **更新配置**

   - 更新 docs routeBasePath 为 `notes`
   - 添加 projects 插件实例

   - 更新搜索插件配置

3. **清理**
   - 删除 `blog/` 文件夹

4. **验证**
   - 启动开发服务器
   - 确认导航正常
