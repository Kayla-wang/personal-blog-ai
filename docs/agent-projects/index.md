## 项目目录

### 一、通用智能体项目

> **特点**：能力全面、适应性强、灵活性高、可处理未知任务

| 项目 | 难度 | 技术栈 | 预计时间 |
|------|------|--------|----------|
| [个人 AI 助理](./01-general/01-personal-assistant/) | ⭐⭐ | LangChain.js + React | 2 周 |
| [通用办公助手](./01-general/02-office-assistant/) | ⭐⭐⭐ | LangGraph.js + Nest.js | 3 周 |
| [多任务自动化引擎](./01-general/03-automation-engine/) | ⭐⭐⭐⭐ | Multi-Agent + MCP | 4 周 |

#### 1.1 个人 AI 助理
- **功能**：日程管理、待办提醒、信息检索、生活服务、旅行规划
- **学习目标**：Agent 基础架构、工具调用、对话管理
- **技术要点**：
  - 意图识别与任务路由
  - 外部 API 集成（日历、天气、地图）
  - 上下文记忆管理

#### 1.2 通用办公助手
- **功能**：文档处理、会议纪要、邮件管理、项目协调、数据整理
- **学习目标**：RAG 技术、多模态处理、流程编排
- **技术要点**：
  - 文档解析与知识库构建
  - 邮件/日历 API 集成
  - 任务优先级排序

#### 1.3 多任务自动化引擎
- **功能**：跨系统操作、数据整合、流程编排、多工具协同
- **学习目标**：MCP 协议、多 Agent 协作、复杂流程控制
- **技术要点**：
  - MCP Server 开发
  - 跨平台工具封装
  - 任务依赖管理与并行执行

### 二、工作流型智能体项目

> **特点**：流程明确、步骤可拆解、结果可预期、自动化程度高

| 项目 | 难度 | 技术栈 | 预计时间 |
|------|------|--------|----------|
| [后端自动化平台](./02-workflow/01-backend-automation/) | ⭐⭐⭐ | Node.js + Cron | 3 周 |
| [前端自动化工具](./02-workflow/02-frontend-automation/) | ⭐⭐⭐ | Playwright + AI | 3 周 |
| [企业审批流程引擎](./02-workflow/03-approval-engine/) | ⭐⭐⭐⭐ | LangGraph.js + 状态机 | 4 周 |

#### 2.1 后端自动化平台
- **功能**：数据同步、报表生成、邮件发送、定时任务、数据库备份
- **学习目标**：定时任务、数据管道、异常处理与重试
- **技术要点**：
  - Cron 任务调度
  - 数据 ETL 流程
  - 告警与日志系统

#### 2.2 前端自动化工具
- **功能**：表单自动填写、页面操作、测试用例生成、UI 自动化测试
- **学习目标**：浏览器自动化、AI 辅助测试、视觉回归
- **技术要点**：
  - Playwright/Puppeteer 操作
  - AI 生成测试脚本
  - 截图对比与异常检测

#### 2.3 企业审批流程引擎
- **功能**：审批流程自动化、工单处理、客户服务流程、入职流程自动化
- **学习目标**：状态机设计、流程可视化、人机协作
- **技术要点**：
  - BPMN 流程建模
  - 条件分支与并行网关
  - 审批节点与 AI 辅助决策

### 三、垂类智能体项目

> **特点**：领域专业性强、准确率要求高、定制化程度高、需要领域知识

| 项目 | 难度 | 技术栈 | 预计时间 |
|------|------|--------|----------|
| [代码助手](./03-vertical/01-code-assistant/) | ⭐⭐⭐ | LangChain.js + AST | 3 周 |
| [医疗问诊助手](./03-vertical/02-medical-assistant/) | ⭐⭐⭐⭐ | RAG + 知识图谱 | 4 周 |
| [金融分析助手](./03-vertical/03-finance-assistant/) | ⭐⭐⭐⭐ | 数据分析 + 可视化 | 4 周 |
| [教育辅导助手](./03-vertical/04-education-assistant/) | ⭐⭐⭐ | 自适应学习 + 评估 | 3 周 |

#### 3.1 代码助手
- **功能**：代码生成、代码审查、Bug 修复、文档生成、项目脚手架
- **学习目标**：代码理解、AST 操作、上下文工程
- **技术要点**：
  - 代码库索引与检索
  - Diff 生成与应用
  - 代码质量规则集成

#### 3.2 医疗问诊助手
- **功能**：问诊咨询、病历分析、用药指导、健康管理
- **学习目标**：医学知识库、安全边界、隐私保护
- **技术要点**：
  - 医学知识图谱构建
  - 症状-疾病推理
  - 严格的输出约束与免责声明

#### 3.3 金融分析助手
- **功能**：数据分析、风险评估、投资建议、财务报表生成
- **学习目标**：金融数据处理、合规要求、可解释性
- **技术要点**：
  - 金融数据 API 集成
  - 风险模型与评分
  - 报表自动生成与可视化

#### 3.4 教育辅导助手
- **功能**：个性化学习、作业批改、答疑解惑、课程推荐
- **学习目标**：自适应学习、知识点追踪、学习路径规划
- **技术要点**：
  - 知识点图谱与掌握度模型
  - 题目难度自适应
  - 学习进度分析与建议

## 项目结构模板

每个项目文件夹包含以下内容：

```
project-name/
├── index.md              # 项目概述与目标
├── 01-requirements.md    # 需求分析
├── 02-architecture.md    # 架构设计
├── 03-implementation.md  # 核心实现
├── 04-testing.md         # 测试方案
├── 05-deployment.md      # 部署指南
└── 06-summary.md         # 总结与优化
```

## 技术选型建议

| 场景 | 推荐方案 |
|------|----------|
| 简单对话 Agent | LangChain.js + Vercel AI SDK |
| 复杂流程控制 | LangGraph.js |
| 多 Agent 协作 | LangGraph.js + Supervisor 模式 |
| 知识库问答 | RAG + 向量数据库 |
| 外部工具集成 | MCP 协议 |
| 企业级部署 | Nest.js + Docker + PM2 |

## 学习建议

1. **由简入繁**：从个人助理开始，逐步挑战复杂项目
2. **完整闭环**：每个项目都要完成从需求到部署的全流程
3. **文档先行**：先写设计文档，再写代码
4. **持续迭代**：MVP 先上线，再持续优化

---

> 💡 **提示**：选择一个感兴趣的领域深耕，比广泛涉猎更有价值。
